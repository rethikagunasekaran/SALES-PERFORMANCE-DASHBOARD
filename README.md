# Sales Performance Dashboard (Excel)

## Abstract
This project presents the design, development, and analysis of a **Sales Performance Dashboard** created using **Microsoft Excel**.  
The dashboard transforms raw sales data into meaningful business insights using **Pivot Tables, charts, and interactive slicers**.

The objective is to support **data-driven decision-making** by presenting sales and profit performance across regions, customer segments, and product categories in a clear and professional manner.

## 6. Methodology

### 6.1 Data Preparation
- Reviewed dataset for missing or inconsistent values
- Removed unnecessary columns
- Ensured correct data types for numeric and categorical fields

### 6.2 Data Analysis Using Pivot Tables
Multiple Pivot Tables were created to summarize data:
- Region-wise Sales and Profit
- Segment-wise Sales and Profit
- Category and Sub-Category performance

Aggregation functions such as **SUM** were used to calculate total sales and total profit.

### 6.3 Data Visualization
Pivot Charts were created from the Pivot Tables:
- Column charts for region and segment analysis
- Bar charts for category-level comparison

Charts were formatted with clear titles, labels, and consistent design.

### 6.4 Dashboard Creation
- All charts were arranged on a single dashboard sheet.
- Slicers were added for **Region, Segment, and Category**.
- Enabled interactive filtering of data.
